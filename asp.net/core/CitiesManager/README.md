# 🏙️ Cities Manager API

A practical implementation of a RESTful API using **C#** and **ASP.NET Core**. This project acts as the backend engine for my full-stack integration tests.

## 🔗 Related Projects
* 💻 **Frontend:** The Angular 20 implementation for this API is located in [Angular Lab - Cities Client](https://github.com/jcdotnet/angular-lab/tree/main/cities-client-app).

## 🚀 C# & Security Focus
* **ASP.NET Core Identity:** Custom implementation using `ApplicationUser` and `ApplicationRole` for robust user management.
* **JWT Authentication:** Secure token-based access control (`JwtBearer`) for stateless communication with the frontend.
* **Identity & GUIDs:** Handling unique identification across the full stack.
* **Modern C#:** Native Dependency Injection and Entity Framework Core for data persistence.

## 🚦 Key Features
* **Secure REST API:** Protected endpoints with authorization policies.
* **CORS Enabled:** Configured for seamless integration with the Angular client.
* **Data Access:** Direct integration with **Entity Framework Core** and SQL Server.
